알아두면 좋을 자바스크립트와 타입스크립트, 그리고 리액트에서 사용할 수 있는 최신 문법 10가지를 정리해보았습니다. 이 리스트는 주로 ES6 이후에 도입된 문법들을 중심으로 구성되어 있으며, 현대적인 자바스크립트 및 타입스크립트 개발에 있어 필수적인 요소들을 포함하고 있습니다.

## 자바스크립트 & 타입스크립트 최신 문법 10가지

### 1. **Template Literals (템플릿 리터럴)**
```javascript
// ES6 이전
var name = "JavaScript";
console.log("Hello, " + name + "!");

// ES6 이후
const name = "JavaScript";
console.log(`Hello, ${name}!`);
```
동적인 문자열을 생성할 때 `+` 연산자 대신 백틱(`)과 `${}`을 사용하여 보다 직관적으로 표현할 수 있습니다.

### 2. **Arrow Functions (화살표 함수)**
```javascript
// ES6 이전
function add(a, b) {
  return a + b;
}

// ES6 이후
const add = (a, b) => a + b;
```
보다 간결한 함수 표현이 가능해집니다. `this` 바인딩에 있어서도 일반 함수와 다르게 동작하여 유용하게 사용됩니다.

### 3. **Destructuring Assignment (구조 분해 할당)**
```javascript
const person = {name: "John", age: 30};
// ES6 이전
var name = person.name;
var age = person.age;

// ES6 이후
const {name, age} = person;
```
객체 또는 배열의 속성을 해체하여, 그 값을 개별 변수에 담을 수 있습니다.

### 4. **Spread Operator (전개 연산자)**
```javascript
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5]
```
배열이나 객체의 속성을 쉽게 복사하거나 합칠 수 있습니다.

### 5. **Async/Await**
```javascript
async function fetchData() {
  const data = await fetch("https://api.example.com");
  console.log(data);
}
```
비동기 코드를 동기 코드처럼 보이게 하여 가독성을 향상시킵니다.

### 6. **Optional Chaining (?.)**
```javascript
const person = {name: "John", address: {city: "New York"}};
console.log(person.address?.city); // "New York"
console.log(person.profile?.age); // undefined
```
객체의 속성에 접근할 때, 해당 속성이 존재하지 않아도 오류를 발생시키지 않고 `undefined`를 반환합니다.

### 7. **Nullish Coalescing Operator (??)**
```javascript
const input = "";
const output = input ?? "default";
console.log(output); // ""
```
왼쪽 피연산자가 `null` 또는 `undefined`일 때만 오른쪽 피연산자를 반환합니다.

### 8. **ES Modules (ES 모듈)**
```javascript
// lib.js
export const add = (a, b) => a + b;

// app.js
import { add } from './lib.js';
console.log(add(1, 2)); // 3
```
모듈 시스템을 사용하여 코드를 여러 파일로 분리하고 재사용할 수 있습니다.

### 9. **TypeScript: Optional Types (선택적 타입)**
```typescript
function greet(name: string, age?: number) {
  console.log(`Hello, ${name}!`);
}
```
타입스크립트에서는 변수나 함수의 인자에 타입을 지정할 수 있으며, `?`를 사용하여 선택적 매개변수를 표시할 수 있습니다.

### 10. **React: Hooks (리액트 훅)**
```javascript
import React, { useState, useEffect } from 'react';

function ExampleComponent() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `You clicked ${count} times`;
  });

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}
```
리액트 16.8 이후에 도입된 훅을 사용하여 함수 컴포넌트에서 상태 관리 및 생명주기 기능을 사용할 수 있습니다.

이러한 문법들을 익히고 활용하면 현대적인 자바스크립트 및 타입스크립트 개발에 있어 보다 효율적이고 간결한 코드를 작성할 수 있습니다.
